## 算法讲解003 二进制和位的运算
### 对于二进制的识别
其中二进制的数据分为有符号的和无符号的。其中无符号的就表示其代表正数。  
如果是有符号的数据就分为正数和负数：  
正数的二进制的首位数字是0，而负数的二进制的首位数字是1。
### 二进制和十进制的相互转化
有关正负的十进制对于二进制的转化：  
正的十进制的转化:  
负的十进制的转化的步骤：  
首先将负数变正->正数转化为二进制->二进制数再减一->减一后的二进制数再取其相反数
示例：
```java
-2->2->0010->0001->1110
-7->7>0111->0110->1001
```
### 怎么从二进制转到对应的十进制
对于带符号的二进制数：  
首先对二进制数取反->然后对二进制数加1->然后这个二进制数对应的十进制数就是真实对应数字的相反数   
示例：
```java
1000->0111->1000->8->-8
```
